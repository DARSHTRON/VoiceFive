# How To Install?
- `git clone https://github.com/DARSHTRON/VoiceFive`
- `cd VoiceFive`
- `pip3 install -r requirements.txt` or you can build a new venv
- Edit the token in `main.py`
- `python3 main.py`
and that's it!, enjoy :>
